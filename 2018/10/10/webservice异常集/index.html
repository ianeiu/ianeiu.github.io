WebService——异常集

作者： 吴炜棉 / 2018-10-10 / 分类： WebService

异常

客户端 javax.xml.ws.soap.SOAPFaultException：MustUnderstand headers

异常： javax.xml.ws.soap.SOAPFaultException: MustUnderstand headers: [{http://docs.oasis-open.org/wss/2004/01/oasis-200401-wss-wssecurity-secext-1.0.xsd}Security] are not understood.

解决：新增 outProps.put(“mustUnderstand”, “0”);

code：

public static <T> T getService(String address, String identifier, String token, Class<T> serviceEndpointInterface) {
    Map<String, Object> outProps = new HashMap();
    outProps.put("action", "UsernameToken");
    outProps.put("user", identifier);
    outProps.put("passwordType", "PasswordDigest");
    outProps.put("passwordCallbackRef", new WSCallbackHandler(identifier, token));
    //新增免验证
    outProps.put("mustUnderstand", "0");
    ArrayList<Interceptor<? extends Message>> list = new ArrayList();
    list.add(new SAAJOutInterceptor());
    list.add(new WSS4JOutInterceptor(outProps));
    JaxWsProxyFactoryBean factory = new JaxWsProxyFactoryBean();
    factory.setServiceClass(serviceEndpointInterface);
    factory.setAddress(address);
    factory.getOutInterceptors().addAll(list);
    return (T) factory.create();
}
