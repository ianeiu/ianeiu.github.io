Oracle——异常集

作者： 吴炜棉 / 2018-09-19 / 分类： Oracle

Oracle异常

同网段计算机无法访问本地数据库

现象：同网段计算机无法访问本地数据库,telnet 192.168.1.101 1521 不通
解决：
修改本地 F:\oracle\wm\product\11.2.0\dbhome_1\NETWORK\ADMIN 目录下的 listener.ora 及 tnsnames.ora 将 HOST = localhost 改成 HOST = wm-PC【计算机名】
重启本地oracle的service及listener服务

安装异常 INS-13001

win10系统安装oracle11g时遇到INS-13001环境不满足最低要求

打开database->stage->cvu->cvu_prereq.xml，编辑内容

<OPERATING_SYSTEM RELEASE="6.2">
    <VERSION VALUE="3"/>
        <ARCHITECTURE VALUE="32-bit"/>
        <NAME VALUE="Windows 10"/>
        <ENV_VAR_LIST>
            <ENV_VAR NAME="PATH" MAX_LENGTH="1023" />
        </ENV_VAR_LIST>
</OPERATING_SYSTEM>

这一段和原先不同就是6.2和window10。如果你系统安装的是64位的，则将32-bit改成64bit即可，然后重新安装。

连接报错 ORA-12505

使用SQL Developer 连接DB报错 ORA-12505，报错信息如下：
ORA-12505, TNS:listener does not currently know of SID given in connect descriptor

解决：找到Oracle安装目录（类似D:\app\mian\product\11.2.0\dbhome_1\NETWORK\ADMIN ）下的tnsnames.ora和listener.ora文件，备份下，然后对照下面配置进行修改

tnsnames.ora

# tnsnames.ora Network Configuration File: D:\app\mian\product\11.2.0\dbhome_1\network\admin\tnsnames.ora
# Generated by Oracle configuration tools.

LISTENER_ORCL =
  (ADDRESS = (PROTOCOL = TCP)(HOST = DESKTOP-UKQ7RTS)(PORT = 1521))


ORACLR_CONNECTION_DATA =
  (DESCRIPTION =
    (ADDRESS_LIST =
      (ADDRESS = (PROTOCOL = IPC)(KEY = EXTPROC1521))
    )
    (CONNECT_DATA =
      (SID = CLRExtProc)
      (PRESENTATION = RO)
    )
  )

ORCL =
  (DESCRIPTION =
    (ADDRESS = (PROTOCOL = TCP)(HOST = DESKTOP-UKQ7RTS)(PORT = 1521))
    (CONNECT_DATA =
      (SERVER = DEDICATED)
      (SERVICE_NAME = orcl)
    )
  )

listener.ora

# listener.ora Network Configuration File: D:\app\mian\product\11.2.0\dbhome_1\network\admin\listener.or
# Generated by Oracle configuration tools.

SID_LIST_LISTENER =
  (SID_LIST =
    (SID_DESC =
      (SID_NAME = CLRExtProc)
      (ORACLE_HOME = D:\app\mian\product\11.2.0\dbhome_1)
      (PROGRAM = extproc)
      (ENVS = "EXTPROC_DLLS=ONLY:D:\app\mian\product\11.2.0\dbhome_1\bin\oraclr11.dll")
    )
    (SID_DESC =
      (GLOBAL_DBNAME = oracle)
      (ORACLE_HOME = D:\app\mian\product\11.2.0\dbhome_1)
      (SID_NAME = oracle)
    )
  )

LISTENER =
  (DESCRIPTION_LIST =
    (DESCRIPTION =
        (ADDRESS = (PROTOCOL = TCP)(HOST = DESKTOP-UKQ7RTS)(PORT = 1521))
    )
    (DESCRIPTION =
      (ADDRESS = (PROTOCOL = IPC)(KEY = EXTPROC1521))
      (ADDRESS = (PROTOCOL = TCP)(HOST = localhost)(PORT = 1521))
    )
  )

ADR_BASE_LISTENER = D:\app\mian

注：
1. DESKTOP-UKQ7RTS 为计算机名
2. 更改文件后需要重启服务
