Spring——@Value注入static属性

作者： 吴炜棉 / 2018-09-04 / 分类： Spring

@Value, static

@Value注入static属性

方式1：
通过执行set方法进行参数注入

private static String userName;
@Value("${git.userName}")
public void setUserName(String userName) {
	GitUtils.userName = userName;
}

方式2：
通过中间变量赋值

private static String userName; 
@Value("${git.userName}")
private String userNameTmp;
 
@PostConstruct
public void init() {
	userName = userNameTmp;
}
