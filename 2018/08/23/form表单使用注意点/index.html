form表单使用注意点

作者： 吴炜棉 / 2018-08-23 / 分类： html

form, json

  1. 查询页面，有绑定回车事件提交表单的，要在form中加上onsubmit=“return false;“，不然会出现回车查询时页面直接显示JSON数据了

  2. 直接在form标签中加入属性accept-charset=‘UTF-8’,防止中文乱码
